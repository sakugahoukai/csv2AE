# 🔄 csv2AE: AE Time Remap Automation

This is an automation utility designed to bridge the gap between **Clip Studio Paint** and **Adobe After Effects**. It analyzes exported CSV timesheets and generates keyframe data ready to be applied directly to the After Effects Time Remap property.

---

### **`[About This Tool]`**

* **Core Function:** Converts Clip Studio CSV timesheets into After Effects Time Remap keyframes.
* **Smart Parsing:** Automatically handles and restores corrupted characters (like `œ`, `›`, `~`) from different language environments.
* **Efficiency:** Eliminates manual keyframe entry, ensuring perfect timing synchronization.

---

### **`[How to Use in After Effects]`**

1.  **Prepare Composition:** Create a composition with your desired size and duration. (Starting timecode **'1'** is recommended.)
2.  **Import Assets:** Add your animation cell folders to the Project panel, then drag them into the Composition panel.
3.  **Run Script:** Select the composition containing the animation cells in the Project panel. Then, go to `File > Scripts > Run Script File...` and select the **.jsx** file generated by **csv2AE**.
4.  **Verify Result:** Success if **Time Remap** is enabled, **Hold Interpolation** is applied, and the layer bars are filled to the end of the composition.

---

### **`[Recommended File Naming]`**

* **Best Practice:** Use the **`'Animation Folder Name + Cell Name'`** convention when exporting animation cells from Clip Studio.
* **Alternative:** The **`'Animation Folder Name + Sequential Number'`** format is also supported.
