# 🔄 csv2AE: 24fps Anime Automation Tool

This tool is optimized for **24fps Anime production** in Adobe After Effects. It analyzes CSV timesheets exported from Clip Studio Paint and generates automation scripts (.jsx) for Time Remapping.

---

### **`[Target Usage]`**
* Optimized for **24fps animation** workflows.
* Bridges Clip Studio Paint timesheets with After Effects timelines.

### **`[CSV Naming & Rules]`**
* **Recommended:** Use the **'Animation Folder Name + Cell Name'** convention when exporting from Clip Studio.
* **Alternative:** 'Animation Folder Name + Sequential Number' is also supported.
* **Note:** Labels like `●` and `○` are automatically processed to hold previous frame properties.

---

### **`[How to Use in After Effects]`**

1. **Prepare Composition** Create a composition with your desired size and duration. (Starting timecode **'1'** is highly recommended.)

2. **Import Assets** Add your animation cell folders to the **Project panel**, then drag them into the **Timeline/Composition panel**.

3. **Run Script** Select the target composition/layer in the Project panel. Go to `File > Scripts > Run Script File...` and select the **.jsx** file generated by this tool.

4. **Verify Result** The process is successful if **Time Remap** is enabled with **Hold Interpolation**, and the layer bars are fully filled to the composition duration.
