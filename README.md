# csv2AE
Clip Studio CSV to After Effects Time Remap (Powered by Gemini)

---

**`[About This Tool]`**

This is an automation utility designed to bridge the gap between **Clip Studio Paint** and **Adobe After Effects**. It analyzes exported CSV timesheets and generates keyframe data ready to be applied directly to the After Effects Time Remap property.

---

**`[Recommended File Naming]`**

* **Recommended:** Use the **'Animation Folder Name + Cell Name'** convention when exporting from Clip Studio.
* **Alternative:** **'Animation Folder Name + Sequential Number'** is also supported.

---

**`[How to Use]`**

Simply drag and drop your CSV file into the area below, or click to upload. The tool will instantly generate the code for your After Effects timeline.
